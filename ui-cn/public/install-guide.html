第一步：下载安装包

获取 ClawbotCN Windows 便携版

1

下载安装包

从官方渠道下载 ClawbotCN-Windows-Portable.zip（约 150MB）

2

检查系统要求

确保已安装 Node.js 22.0 或更高版本

💡
如尚未安装 Node.js，请访问 nodejs.org 下载 LTS 版本
🚀

第二步：解压并启动

运行安装脚本，自动打开配置页面

1

解压安装包

右键点击 ZIP 文件 → 选择「解压到当前文件夹」

2

运行 setup.bat

双击运行 setup.bat 文件

3

访问配置页面

浏览器将自动打开，或手动访问 http://localhost:18789/setup

⚙️

创建飞书应用

登录 飞书开放平台 → 创建企业自建应用

2

获取应用凭证

在「凭证与基础信息」页面获取 App ID 和 App Secret

3

配置机器人能力

进入「添加应用能力」→ 启用「机器人」功能

4

配置事件订阅

添加「接收消息」事件，设置回调地址为 http://你的地址:3001/feishu/webhook

5

发布应用

创建钉钉应用

登录 钉钉开放平台 → 创建企业内部应用

2

获取应用凭证

在「应用凭证」页面获取 AppKey 和 AppSecret

3

配置机器人

进入「应用功能」→「机器人」→ 开启并配置

4

配置消息接收地址

选择 HTTP 模式，设置地址为 http://你的地址:3002/dingtalk/webhook

5

发布应用

在「版本管理与发布」中点击「上线」并设置可用范围

在钉钉开放平台「应用凭证」页面获取
