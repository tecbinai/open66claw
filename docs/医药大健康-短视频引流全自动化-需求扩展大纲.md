## 二、M0 AI 智能选品（新增核心模块）

### 2.1 业务痛点

> "做什么品、卖什么品"决定了整个链路的天花板。人工选品慢、主观、容易踩坑。

### 2.2 选品全流程

```
S1 多平台数据采集
    抖音/快手/小红书/TikTok Shop → 品类销量 + 趋势 + 竞争度
        ↓
S2 多维度筛选建库
    平台 × 品类 × 价格带 × 利润率 × 竞争密度 × 增长趋势
        ↓
S3 品类垂直库
    形成可持续更新的"品类知识库"，按赛道细分管理
        ↓
S4 选品决策
    AI 评分排序 + 人工终审 → 确定主推品 + 引流品 + 利润品
        ↓
S5 供应商匹配
    平台/渠道供应商库 → 价格比对 → 样品测评 → 锁定供应链
```

### 2.3 选品筛选维度

| 维度 | 说明 | 数据源 |
|------|------|--------|
| **品类热度** | 搜索量趋势、话题热度、关联视频播放量 | 平台搜索 + 话题 API |
| **竞争密度** | 同品类带货账号数、视频数、头部集中度 | 平台数据 |
| **利润空间** | 供货价 vs 零售价，毛利率 ≥40% 为佳 | 供应商报价 |
| **合规风险** | 广告法/平台规则限制程度（医药类尤其重要）| 规则库 |
| **内容适配** | 是否容易做视频展示、是否有"种草点" | 人工标注 + AI 分析 |
| **复购潜力** | 消耗品/疗程品优先，单次购买品降权 | 品类属性 |
| **季节性** | 是否受季节/节气影响，提前布局 | 历史数据 |

### 2.4 品类垂直库结构

```
品类垂直库
├── 养生滋补（枸杞/黄芪/阿胶/燕窝/…）
├── 食疗调理（祛湿茶/养胃粉/助眠膏/…）
├── 日常保健（维生素/益生菌/鱼油/…）
├── 中医理疗（艾灸/刮痧/泡脚包/…）
├── 个护健康（洗鼻器/按摩仪/筋膜枪/…）
└── 功能食品（代餐/膳食纤维/酵素/…）

每个品类维护：
  - 市场规模 & 增长趋势
  - TOP 竞品分析
  - 供应商清单 & 价格带
  - 爆款视频案例
  - 合规注意事项
  - AI 推荐评分（动态更新）
```

## 三、M0.5 供应链管理（新增）

### 3.1 核心功能

| 功能 | 说明 |
|------|------|
| 供应商库 | 按品类/渠道/平台管理供应商信息，支持评分和对比 |
| 价格监控 | 多渠道比价（1688/拼多多/厂家直供），自动提醒价格波动 |
| 样品管理 | 样品申请→测评→打分→入库/淘汰，形成闭环 |
| 库存预警 | 基于销量预测的智能补货提醒 |
| 发货对接 | 一件代发/自有仓储的订单对接（可选） |

### 3.2 供应商评估标准

| 维度 | 权重 | 标准 |
|------|------|------|
| 产品资质 | 30% | 生产许可/检测报告/商标授权齐全 |
| 价格竞争力 | 25% | 同品类前 30% 价格带 |
| 供货稳定性 | 20% | 日发货能力、断货率 |
| 响应速度 | 15% | 样品寄送、问题处理时效 |
| 退换货政策 | 10% | 售后支持力度 |

## 四、AI 内容工厂（现有 M3 的 AI 升级）

### 4.1 对标 TikTok 美区 AI 打法

> 美区头部 MCN 已实现：**AI 选题 → AI 脚本 → AI 数字人/口播 → AI 混剪 → 一键多版本 → 自动发布**，单人管理 50+ 账号，成本降低 80%+。

### 4.2 AI 视频生产能力矩阵

| 能力 | 说明 | 参考工具 |
|------|------|----------|
| **爆款一键复刻** | 输入爆款视频链接 → AI 分析结构 → 替换素材/文案 → 生成新版本 | CreatOK / ClipMove |
| **AI 数字人口播** | 上传真人素材训练 → AI 生成口播视频（多情绪、多风格） | HeyGen / D-ID |
| **AI 混剪引擎** | 素材库 + 脚本 → 自动剪辑 + 字幕 + BGM + 转场 | 万媒一发 / 融意引擎 |
| **图文卡片生成** | 文案 + 模板 → 自动排版（小红书风格） | Canva AI / 自建 |
| **多版本差异化** | 同一内容一键生成 5~20 个差异版本（去重+防搬运检测） | 自建 |
| **AI 配音** | 多音色 + 多情绪 + 方言（医药行业需权威感 + 亲和力） | 自建 / TTS |
| **合规自动审核** | 发布前 AI 扫描违禁词/敏感画面/音频违规 | 自建 |

### 4.3 素人二创工作流（核心差异化）

```
S1 爆款逆向拆解
    输入爆款视频 → AI 提取：选题角度 / 内容结构 / 钩子句 / 标签组合 / BGM
        ↓
S2 素人风格迁移
    从素人库中匹配风格 → 口语化改写 → 生活场景替换
        ↓
S3 AI 二创生成
    新脚本 + 素人素材/AI数字人 + 差异化处理 → 批量成片
        ↓
S4 去重检测
    MD5 + 感知哈希 + 平台查重预检 → 确保每个版本独一无二
```

## 五、智能互动引流（现有 M4~M7 的 AI 升级）

### 5.1 AI 评论系统升级

| 升级项 | 说明 |
|--------|------|
| AI 实时生成评论 | 基于视频内容理解，生成与视频高度相关的"神评论"，而非模板库轮换 |
| 情感适配 | AI 判断视频情绪基调，匹配评论风格（严肃/搞笑/共鸣/专业） |
| 上下文互动 | 评论回复 + 追评，模拟真实用户对话链 |
| A/B 测试 | 同一视频下测试不同风格评论，数据反馈优化话术策略 |

### 5.2 AI 私信转化升级

| 升级项 | 说明 |
|--------|------|
| 意图识别 | AI 分析用户消息意图（咨询/购买/犹豫/拒绝），匹配最优回复 |
| 多轮对话 | 支持 3~5 轮自然对话，逐步建立信任后引导加微 |
| 人机协作 | AI 处理标准场景，高意向用户自动转人工客服 |

## 六、数据闭环与智能决策（新增）

### 6.1 全链路数据看板

```
选品数据   →  品类热度/竞争/利润变化
    ↕
内容数据   →  各模板/选题/风格的播放/互动/转化率
    ↕
引流数据   →  各渠道/话术/时段的引流效率
    ↕
成交数据   →  品类/客单价/复购率/ROI
    ↕
       全链路归因 → 哪个品 + 哪种内容 + 哪个渠道 = 最优 ROI
```

### 6.2 AI 决策建议

| 场景 | AI 输出 |
|------|---------|
| 选品推荐 | "祛湿茶品类本周搜索量↑35%，竞争度中等，建议加大投入" |
| 内容优化 | "反常识型模板本周转化率最高，建议增加比例至 40%" |
| 账号预警 | "账号 C3 连续 3 天播放量 <200，疑似降权，建议暂停" |
| 时段优化 | "周三晚 20:00 发布的评论截留效果比 19:00 高 22%" |

## 七、模块优先级建议

> 建议分期交付，快速验证 MVP 后再扩展

| 优先级 | 模块 | 理由 |
|--------|------|------|
| **P0 必做** | AI 选品 + 品类垂直库 | 选品决定天花板，是所有后续动作的前提 |
| **P0 必做** | AI 视频生产（爆款复刻 + 混剪 + 差异化） | 内容是流量核心，AI 化后效率提升 10 倍 |
| **P0 必做** | 评论截留 + 点赞引流 | 引流核心动作，直接产生私域进粉 |
| **P1 重要** | 供应链管理 | 选品确定后需要高效对接供应商 |
| **P1 重要** | AI 数字人口播 | 解决真人出镜成本高、产能受限的问题 |
| **P1 重要** | 数据看板 + 全链路归因 | 优化 ROI 的基础设施 |
| **P2 增强** | AI 私信转化 | 提升人效，但需注意平台风控 |
| **P2 增强** | AI 实时评论生成 | 替代话术库，提升评论质量和自然度 |
| **P3 远期** | AI 决策建议 | 数据积累足够后才有意义 |
| **P3 远期** | 库存预警 + 自动补货 | 规模化后的供应链优化 |

## 八、TikTok 美区 MCN 的 AI 打法参考

> 以下为美区头部带货公司已验证的模式，可选择性借鉴到国内打法

| 打法 | 做法 | 国内适配 |
|------|------|----------|
| **AI 选品雷达** | 实时扫描 TikTok Shop 各品类 GMV/增长率/竞争度，AI 推荐蓝海品 | 可对标抖音/快手电商数据 |
| **AI 视频工厂** | 产品图 → AI 自动生成带货视频（含口播、场景展示、卖点文案），单人日产 100+ 视频 | 完全可复用 |
| **爆款克隆** | 一键分析竞品爆款视频 → 替换产品/人物/文案 → 生成合规新版本 | 核心能力，必须有 |
| **AI 合规预审** | 发布前 AI 扫描视频/文案违规风险，MCN 自建审核系统 | 医药行业更需要 |
| **数据驱动迭代** | 每 24h 分析各视频数据 → AI 建议调整选题/封面/发布时间 | 可复用 |
| **达人矩阵 SaaS** | 统一管理数百达人账号，自动分配任务/素材/排期 | 对应 M8 账号矩阵 |
| **佣金优化** | AI 分析各品类佣金率 vs 转化率，自动调整推广策略 | 带货场景可用 |
