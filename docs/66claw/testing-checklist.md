## 一、前置准备工作

### 1.1 环境要求

| 项目 | 要求 | 检查命令 |
|------|------|---------|
| Node.js | 22+ | `node -v` |
| pnpm | 最新版 | `pnpm -v` |
| Rust toolchain | stable（桌面端测试需要） | `rustc --version` |
| macOS 版本 | 13.0+ | `sw_vers` |
| 磁盘空间 | 至少 2GB 可用 | `df -h .` |

### 1.2 构建项目

```bash
# 1. 安装依赖
pnpm install

# 2. 构建后端 + CN 前端
pnpm build
# pnpm build 末尾自动执行 pnpm ui-cn:build，输出到 dist/control-ui/

# 3. 确认构建产物存在
ls dist/entry.js          # 后端入口
ls dist/control-ui/index.html  # CN 前端
```

### 1.3 确保无残留进程

```bash
# 杀掉可能残留的 gateway 进程
pkill -f "entry.js gateway" || true
pkill -f "openclawcn.mjs gateway" || true

# 确认端口空闲
lsof -ti :18789 | xargs kill -9 2>/dev/null || true
lsof -ti :19002 | xargs kill -9 2>/dev/null || true
```

### 1.4 清理旧配置（可选，干净测试时执行）

```bash
# 备份现有配置
cp ~/.openclaw/openclaw.json ~/.openclaw/openclaw.json.bak 2>/dev/null || true

# 如需全新测试，删除配置（谨慎）
# rm ~/.openclaw/openclaw.json
```

### 1.5 准备测试用 API Key

至少准备以下其中一个 AI 提供商的 API Key：

| 提供商 | 用途 | 备注 |
|--------|------|------|
| DeepSeek | 基础对话 | 国内首选，便宜 |
| 硅基流动 (SiliconFlow) | 图片/视频生成 | 支持多模型 |
| 火山引擎 | 语音 ASR/TTS | 语音测试需要 |
| OpenAI / Anthropic | 高级对话 | 可选 |
